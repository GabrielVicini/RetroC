### What is this?
This project serves as a private runtime for my custom ecosystem of software utilities, handling low-level hardware control and real-time flight calculations. It is currently in a pre-alpha state and **is not yet, nor recommended for public use**, though I plan to provide a more detailed technical breakdown as the architecture matures.
